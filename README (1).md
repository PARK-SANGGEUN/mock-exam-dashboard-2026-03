# 2026년 3월 학력평가 상담용 분석 시스템 | 강원사대부고

강원사대부고 3학년부 상담용 웹입니다.

포함 기능
- 반 학생 전체 표에서 과목별 원점수와 변환 등급 동시 확인
- 학번 / 이름 검색
- 반 필터
- 현재 보이는 목록 엑셀 다운로드
- 학생 클릭 시 개별 상세 화면
- 응시 과목 전체 표시
- 가장 강한 영역 / 가장 보완이 필요한 영역 자동 표시
- 과목별 등급대 내 상대적 위치 시각화
- 등급컷 참고표 탭 제공
- 상단 출처 문구 표시: EBSi 등급컷 기반

사용 방법
1. GitHub 저장소에 `index.html` 업로드
2. Settings → Pages에서 배포 브랜치 선택
3. 접속 후 학번/이름 검색, 반 필터, 학생 클릭으로 상담 화면 활용

참고
- 9등급은 8등급 컷 미만으로 처리
- 등급컷은 제공된 EBSi 이미지 기준 원점수 컷 반영
